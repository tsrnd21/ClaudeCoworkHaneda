§4 · 構造特記事項・解釈上の注意点
非ゾーンセクション (8〜13号)
運搬費(8号)・事業損失防止施設費(9号)・技術管理費(10号)は共通仮設費に属するが、内訳書ではH01レベルとして独立配置
その他(11号)ページ内に地質調査・六価クロムがA列に出現 → H01_ZONESへの登録必須 (未登録時はZONE_MARKER未検出)
12号・13号の詳細ページが別途存在 — その他ページ内の参照金額はスキップ、詳細ページでleaf収集
Two_Row_Item 直結構造
通常構造: H05_HDR + SPEC (price有) → LEAF
直結構造: H02/H03_HDR + SPEC (price有) → LEAF (運搬費・事業損失防止施設費・技術管理費・地質調査 セクション)
集計構造: H02〜H04_HDR + SPEC (price=None、金額のみ) → BRANCH — 二重計上なし
ページ繰り返し構造
33行周期 PAGE_COUNTER + ZONE_MARKER + COL_HEADER繰り返し (計142ページ)
同一Zoneが複数ページにまたがる → ZONE_MARKER重複出現、H01コンテキストは最後のマーカーを基準
COL_HEADER行 ("費目・工種…") 140回 — パース時に必ずフィルタリング
